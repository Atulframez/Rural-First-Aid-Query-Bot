Rural First Aid Query Bot is a Python-based project that provides quick first-aid guidance for common emergency situations such as bleeding, fractures, burns, electric shock, choking, animal bites, snakebites, fainting, heart attack symptoms, heat stroke, and pregnancy-related emergency care. The bot works offline and is designed for rural and remote regions where instant access to medical help may not always be available. It also provides important Indian emergency helpline numbers to assist users in critical conditions.
